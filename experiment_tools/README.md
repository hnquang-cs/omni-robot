# experiment_tools

## Purpose
This package contains helper scripts for bag recording, result export, and experiment bookkeeping across thesis runs.

## Planned inputs and outputs
- Input: bags, map outputs, odometry logs, and benchmark metadata.
- Output: recorded bag files, exported summaries, and structured experiment folders.

## Important files
- `scripts/record_bag.sh`: placeholder rosbag capture helper.
- `scripts/export_results.py`: placeholder export script.

## Next-stage work
- Standardize bag naming and storage rules.
- Export quantitative benchmark results.
- Add scenario metadata templates for experiments.
